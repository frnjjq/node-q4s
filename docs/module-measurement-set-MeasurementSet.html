Class: MeasurementSet

measurement-set~MeasurementSet(latency, jitterUp, jitterDown, bandwidthUp, bandwidthDown, packetlossUp, packetlossDown)

Represents a complete q4s session quality measures. This class contains methods to serve as reference quality metric. Contains too methods that allow to map server and client measures into a session set of measures.
